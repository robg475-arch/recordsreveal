# Comparative Financial Analysis Skill

Analyzes financial data with A vs B comparisons (campaign spending, budgets, donations, etc.)

## What It Does

Automatically detects and analyzes:

1. **Financial Columns** - Spending, revenue, costs, donations, budgets
2. **Comparative Patterns** - Dem vs Rep, Before vs After, Support vs Oppose
3. **Top Entities** - Highest spenders, top recipients, biggest budgets
4. **Spending Distribution** - Totals, averages, medians, ranges
5. **Comparative Advantage** - Which side has more money, by how much
6. **AI Insights** - Newsworthy findings generated by Ollama

## Usage

### Automatic (via Pipeline)
```bash
# Profile will auto-detect financial data and call this module
./run_full_pipeline.sh campaign_finance.csv
```

### Manual
```bash
python3 .opencode/skills/comparative-financial-analysis/analyze.py \
  data/campaign_finance.csv \
  pipeline_output
```

## What Gets Detected

### Financial Keywords
The module looks for columns containing:
- `spend`, `cost`, `price`, `revenue`, `budget`
- `donation`, `contribution`, `support`, `oppose`
- `money`, `dollar`, `amount`, `total`, `payment`

### Comparative Patterns

**Political:** `dem` vs `rep`, `democrat` vs `republican`
**Temporal:** `before` vs `after`, `pre` vs `post`, `old` vs `new`
**Binary:** `yes` vs `no`, `for` vs `against`, `support` vs `oppose`

## Key Features

### 1. Auto-Detection
No manual column specification - the module finds financial and comparative columns automatically.

### 2. Top Entities
Identifies the top 10 entities by total spending/revenue/budget.

### 3. Distribution Analysis
Calculates totals, averages, medians, and ranges for all financial columns.

### 4. Comparative Advantage
If A vs B pattern detected, calculates which side has more money and by how much.

### 5. AI Insights
Ollama generates newsworthy findings like:
- "Democrats outspent Republicans by $76M across swing districts"
- "Top 10 districts accounted for 45% of all spending"
- "Average spending per district: $3.4M"

## Detection Logic

```python
# Financial column detection
if column_name contains ('spend', 'cost', 'revenue', 'budget', etc.)
   AND column is numeric
   → Mark as financial column

# Comparative pattern detection  
if 2+ columns contain ('dem' and 'rep')
   OR ('before' and 'after')
   OR ('support' and 'oppose')
   → Mark as comparative pattern

# If both detected → Run comparative financial analysis
```
