## How to Use These Prompts

### With ChatGPT (DALL-E):
1. Open ChatGPT (Plus subscription required for DALL-E 3)
2. Copy and paste one of the prompts above
3. ChatGPT will generate the image
4. Download and save as PNG
5. Optimize if needed (use tinypng.com to compress)

### With Other AI Tools:
- **Midjourney**: Add `--ar 2:1` for horizontal format, `--ar 1:1` for square
- **Leonardo.ai**: Select "Illustration" preset, paste prompt
- **Stable Diffusion**: Use "editorial illustration" in negative prompt to avoid photo-realism

### Tips for Best Results:
1. **Try all 3 options** per investigation and pick the best
2. **Iterate**: If the first result isn't perfect, ask to "make it more minimalist" or "use brighter red"
3. **Consistency**: Use the same option style (A, B, or C) across all investigations for cohesion
4. **Format**: Request PNG with transparent background if you want flexibility

### File Naming Convention:
- `hero-bird-strikes.png` (1200x600)
- `card-bird-strikes.png` (800x800)
- `hero-hollywood.png` (1200x600)
- `card-hollywood.png` (800x800)
- etc.
